## derivatives
### second derivative 二阶导数
f'' = (f')'
### convex and concave curves 凸函数和凹函数
convex: if f'' > 0  凸函数 V型  
concave: if f'' < 0 凹函数 ^型

### Inflection Point 拐点
convex 与 concave 相互变换的点， 即 f'' = 0 的时候

### derivative equation
dy/dx = y => e
  

## functions
### y = x^n

## y = sinx

## y = e^x
